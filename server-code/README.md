# RichUp Clone (Lite)

A super lightweight, high-performance Monopoly-style game engine.
Built for low-latency networks and minimal bundle size.

## Tech Stack (Phase 1)
- **Engine:** TypeScript / Node.js
- **State:** Pure JSON/Binary structure (no heavy classes)
- **Sim:** Console-based game loop included

## Run Simulation
```bash
npm install
npx ts-node src/simulation.ts
```
